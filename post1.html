Project: Analyze for Usability

Week 1 Assignment

Posted by Douglas Jones on April 1, 2019

What site are you working with?

The site I am analyzing is Wanikani

What is the purpose of this site

The purpose of this site is to help learners of Japanese to study Kanji. This is a subscription based website that utilizes spaced repetition and mnemonics to help users commit Kanji and vocabulary to memory.

What criteria are you using to analyze the site?

These are the criteria I will be using to analize the site.

  • Is it useful? - Does it do something people need done?
  • Is it learnable? - Can people figure out how to use it?
  • Is it memorable? - Do they have to relearn it each time they use it?
  • Is it effective? - Does it get the job done?
  • Is it efficient? - Does it do it wit ha reasonable amount of time and effort?
  • Is it delightful? - Is using it enjoyable, or even fun?

Is it useful?

I would say that this website is only useful to those who are serious about learning Japanese, as this is a subscription based service.

Is it learnable

It is extremly learnable as there is not much too the website itself besides using it to learn Japanese. You can review past lessons as they appear or continue on to new lessons once you have reached a new level.

Is it memorable

For me it's memorable website with an odd name. "WaniKani" translates to "Aligator Crab" which alone is something to stick in someone's mind especially with all the visual dipections of the WaniKani on the website.

Is it effective

As someone who uses the site, I can say that it is indeed useful, I was able to learn a great deal in such a short amount of time. m

Is it efficient?

For what it does it does it's best to teach its users in an efficient manner by first starting with the more commonly used Kanji and then continuing on to the less used ones.

Is it delightful?

I enjoy using the website. I always like waiting for the reviews to pop up so I can test my knowledge and see if I can remember what I learned.
